## About Me

I’m **Sachida Sharma**, currently working as a **TechOps Engineer at RailTel (RDSO, Indian Railways)**.  
I troubleshoot and secure enterprise systems such as **e-Office, HRMS, CRISP UVAM, and SPARROW**, which sparked my interest in **Security Operations and Blue-Team Defense**.  
I spend my free time running **SOC labs**, monitoring logs, and studying incident-response workflows.

**Current Focus**
- Building home-lab simulations using **IBM QRadar** and **Splunk**  
- Completing **TryHackMe Blue Team** and **SOC Level 1** learning paths  
- Practising packet capture & analysis with **Wireshark / Cisco Packet Tracer**  
- Documenting incidents and threat-hunting notes on GitHub
